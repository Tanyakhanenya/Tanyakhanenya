1. Отобразить подключённый девайс в консоли - ./adb devices
 2. Вывести адрес приложения todolist в системе Android - cd C:\001_ToDoList-master\app\build\outputs\apk\debug
 3. Установить .apk файл приложениия todolist на телефон с компьютера через  ADB - C:\adb> ./adb install C:\001_ToDoList-master\app\build\outputs\apk\debug\Todolist.apk
 4. Сделать скриншот запущенного приложения todolist и сразу скопировать на компьютер в одной команде. - ./adb shell screencap /storage/emulated/0/DCIM/143.png | ./adb pull /storage/emulated/0/DCIM/123.png /adb/143.png
 5. Вывести в консоль логи приложения todolist - ./adb logcat
 6. Скопировать логи приложения todolist на компьютер. - ./adb shell "logcat | grep -rnw todolist" > log_todolist1.log
 7. Удалить приложение todolist с телефона через ADB - ./adb uninstall com.android.todolist
